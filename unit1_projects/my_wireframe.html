The Plan

Apply the Principles of Good Web Design to create an index site that will be easy to expand as more material is added.

Goals:

  1. Create a sidebar that lists projects and blog posts by week. Items in the sidebar link to the previews in the main window.
  2. Display linked previews of the post or project in the main window.
  3. Use a responsive grid so that all elements will display correctly on different screen sizes.
  4. Use "alt" tags for accessibility.

  • Layout

    Give each element room to breathe. Whitespace is important for readability

  • Flow

    Guide the user through the space of the site using images, typography, and lines to divide into sections and draw attention.

  • Alignment

    Use a responsive grid system to organize the space.

  • Navigation

    Highlight the current selection (this might be tricky!)

  • Colors

    Use colors that contrast eachother for readability. Use brighter colors for "call to action" items.

  • Content

    Use appropriate imagery and typography to differentiate sections and guide the flow.
